# Bitmap Text Change Log
All notable changes to this project will be documented in this file.

## [1.1.5] - 2025-01-23
### Fixed
- Basic code issues including missing variables
- Now uses sf::Vector2<std::size_t> instead of sf::Vector2u for consistency

## [1.1.4] - 2025-01-23
### Fixed
- Updated to use SFML version 3
- Update code style

## [1.1.3] - 2023-04-17
### Fixed
- Removed use of deprecated quads primitive, replaced with triangles

## [1.1.2] - 2018-11-07
### Fixed
- Added some missing class members' initialisation
- Updated code style

## [1.1.1] - 2017-07-17
### Fixed
- setBitmapFont can now also take no parameters to nullify font
- Updated code style

## [1.1.0] - 2016-07-29
### Added
- Implementations of getLocalBounds() and getGlobalBounds()

## [1.0.0] - 2015-08-19
### Added
- Full version
